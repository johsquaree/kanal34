- **Özet**: 15px, 1.5 line-height
- **Kategori**: Badge tasarımı

## 📱 Responsive Grid Layout
- **Desktop**: 3 sütun (repeat(3, 1fr))
- **Tablet**: 2 sütun (768px altında)
- **Mobile**: 1 sütun (480px altında)
- **Gap**: 32px aralık
- **Min Height**: 420px kart yüksekliği
